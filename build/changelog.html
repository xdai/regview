Change Log

v1.0.0 6/19/2020

The first feature complete release. Built based on React and Ant Design.

  • Visulized register field editor
  • You can add named constants for each register field
  • Registers are grouped in a tree hierarchy
  • Encoder: You can specify individul field values and get register value as a whole
  • Decoder: You can specify register value as a whole and get individul field values
  • Compare: You can compare two register values and see which fields are different
  • Data can be imported from a JSON file
  • Data can be exported as a JSON file
  • Data can be exported to several different flavors of C/C++ code
  • Data is persisted in IndexedDb in your web browser locally
  • Doesn't need a server (http, database, whatsoever) to run
